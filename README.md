Amazon Web Services CLI tools updater.
===============

Installs CLI tools for
- Autoscaling
- EC2
- RDS
- IAM
- CloudFormation
- CloudWatch
- AMI tools
- Elastic Load Balancer

Creates directory ~/AWS_CLI_TOOLS and updates env variables by ~/.aws_cli_tools

Redownloads tools on each run.
